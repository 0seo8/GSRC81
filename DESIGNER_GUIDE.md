# 그래픽 지도 오버레이 디자이너 가이드

## 📋 개요
실제 Mapbox 지도 위에 그래픽화된 서울 지도 이미지를 오버레이하여, 사용자가 줌/팬 할 때 두 지도가 완벽하게 동기화되도록 구현합니다.

## 🎯 목표
- 실제 지도와 그래픽 지도의 완벽한 동기화
- 줌 레벨별 정확한 매칭
- 모바일/웹 반응형 대응

## 📏 기준 사양

### 지도 중심점
- **위도/경도**: `37.6176, 126.9227` (은평구 중심)
- **기준 줌 레벨**: `12`

### 캡처해야 할 줌 레벨
1. **최소 줌 (Zoom 10)**: 서울 전체 (25개 구 모두 보임)
2. **일반 줌 (Zoom 12)**: 은평구 중심 + 주변 구들 (8-10개 구)
3. **최대 줌 (Zoom 16)**: 상세 지역 (2-3개 구)

### 컨테이너 크기
- **웹**: 가변 크기 (최소 800px 기준)
- **모바일**: 375px ~ 768px

## 🛠️ 캡처 도구 사용법

### 1. 캡처 도구 접근
- 지도 우상단의 "디자이너 캡처 도구" 패널 사용
- 실시간으로 현재 줌/중심점 확인 가능

### 2. 각 줌 레벨별 캡처
```
1. "최소 줌 (서울 전체)" 버튼 클릭
2. "이미지 다운로드" 버튼으로 스크린샷 저장
3. "정보 캡처" 버튼으로 좌표 정보 클립보드 복사
4. 일반 줌, 최대 줌도 동일하게 반복
```

### 3. 캡처된 정보 예시
```json
{
  "zoom": 12,
  "center": [126.9227, 37.6176],
  "bounds": {
    "north": 37.6789,
    "south": 37.5563,
    "east": 127.0234,
    "west": 126.8220
  },
  "containerSize": {
    "width": 1200,
    "height": 800
  }
}
```

## 🎨 그래픽 디자인 요구사항

### 스타일 가이드
- **배경색**: 주황색 계열 (#FF6B35 또는 유사)
- **구 경계선**: 검은색 실선 (2-3px)
- **구 이름**: 각 구 중앙에 배치
- **한강**: 파란색으로 표현
- **전체적**: 각진 형태, 미니멀 디자인

### 파일 형식
- **형식**: PNG (투명 배경 권장)
- **해상도**: 최소 1600x1200px
- **DPI**: 72 또는 144
- **색상 모드**: RGB

### 레이어 구조
```
1. 배경 (투명 또는 주황)
2. 한강 (파란색)
3. 구 영역 (채우기)
4. 구 경계선 (검은 선)
5. 구 이름 텍스트
```

## 🔧 구현 단계

### Phase 1: 기준 이미지 제작

3. 테스트 적용 및 정렬 조정
