## ✅ 코드 반영 리스트 (통화 기반)

### 1. **인트로(랜딩/스플래시) 관련**

- **인트로 애니메이션 항상 표시되도록 변경**
  -> 이유 현재 웹뷰형태로 구현되어있으며 카카오톡으로 링크를 통해 공유한후 그 링크를 타고들어오는 경우가 많을텐데 그런경우 넥플릭스 두둥!하고 뜨는 로고나 디즈니 접속시 뜨는 로고처럼 우리도 스프래시 이벤트라기 보다는 하나의 인트로 애니메이션이 되길 바람. 따라서 진입할때마다 뜨길 바람.

- **카카오 로그인 세션은 유지**
  - 인트로는 항상 새로 표시되지만, 로그인 정보는 유지되어 바로 맵 진입 가능해야 함
  -

- **정렬/폰트 차이 수정**
  - 피그마 기준으로 텍스트 위치, 폰트, 색상, 정렬 등 재확인 필요 (현재 인텔 폰트로 잘못 적용된 상태)
    -> 재정리

- **인트로 화면 UI 검증**
  - 검정 배경, 여백 등 피그마와 불일치 부분 수정

---

### 2. **리스트 및 바텀시트 동작**

- **리스트와 바텀시트 구조 개선**
  - 현재 “리스트와 바텀시트가 분리되어 있음” → **같은 컴포넌트 안에서 함께 동작하도록 통합**
  - 바텀시트 내에서 루트 리스트를 보여주는 구조로 변경

- **바텀시트 레이어 문제 해결**
  - 현재 바텀시트가 뜰 때 **지도 위에 어둡게 레이어(overlay)** 씌워짐
    → 제거 (지도 내용이 계속 보이도록)

- **바텀시트 열림/닫힘 상태**
  - 항상 절반 높이(snap point 50%)에서 시작
  - 위로 당기면 “풀(open)” 상태로 확장, 내리면 다시 절반으로 복귀

- **루프 기능 추가**
  - 현재 map페이지에서 바텀시트가 열린상태에서 옆으로 넘기는 경우 스와이프가 되고 있는데 마지막 슬라이드에서 다시 첫 번째로 돌아가도록 구현 (carousel loop)

- **리스트 카드 마감 처리**
  - 루트 카드가 2개일 때는 하단이 둥근 모서리로 끝나야 함
  - 3개 이상일 경우 마지막 카드도 "스크롤을 내려보면" 자연스럽게 곡선 형태로 마감되도록 수정 스크롤을 내리기전에는 짤려보이는게 맞음

- **여백 및 폰트 크기 조정**
  - 피그마와 실제 화면 간의 margin/padding 불일치 수정
  - 카드 내 텍스트(`거리`, `코스명`, `난이도`)의 줄 간격 및 위치 재조정

---

### 3. **지도(Map) 관련 기능**

- **내 위치 확대 레벨 조정**
  - 현재 너무 멀게 표시됨 → 초기 줌 레벨 상향

- **로딩 상태 개선**
  - 지도 데이터(API) 로딩 지연 시 “전체 러닝”이 먼저 뜨는 현상 수정
  - 대안:
    - API 응답 전에는 **로딩 스켈레톤 or 공백 상태 유지**
    - 실패 시만 “전체 러닝” 표시

- **댓글(코멘트) 기능 개선**
  - 노선 위에서만 댓글 추가 가능하도록 수정
    → 경로 외 클릭 시 `toast` 또는 무반응
  - **길게 누르기(약 2~5초)** 시 진동 피드백 추가
  - 해당 위치에서 “코멘트 추가” 팝업 표시
  - 진동(`navigator.vibrate`) 지원 여부 확인 및 브라우저별 폴백 처리

- **댓글 팝업 개선**
  - 루트 가리지 않도록 위치 조정
  - 팝업 내용: “메시지를 남겨주세요.” 등의 안내 문구 추가

---

### 4. **관리자(Admin) 관련**

- **유저 관리 기능 추가**
  - 관리자 페이지에서 사용자 목록 확인 + 관리자로 지정/해제 가능

- **최고 관리자 개념 추가**
  - 최고 관리자가 다른 관리자를 생성/삭제할 수 있는 구조 필요

- **코스 관리**
  - 코스 등록/삭제, 카테고리 분류 기능 유지
  - 일반 사용자도 추후 “내 루트 업로드” 기능 고려 (마이페이지 연동 가능성 있음)

---

### 5. **기타 UX/UI**

- **바텀시트 슬라이드 애니메이션 자연스럽게 개선**
  - 부드럽게 ‘슉슉’ 넘어가는 전환 효과 (`framer-motion` or `react-spring`)

- **배경 레이어 제거 시 대비 유지**
  - 지도와 바텀시트 간 시각적 구분은 “그림자(shadow)”로 대체

- **폰트 적용 오류 수정**
  - 현재 모든 텍스트가 Inter로 fallback 되는 문제 해결
  - 지정 폰트(`SUIT`, `Pretendard`, `Noto Sans KR`) 다시 설정
